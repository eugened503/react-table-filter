# Тестовое задание «Интерактивная таблица» для «Crazy Panda»

## Описание тестового задания 
Работа представляет собой интерактивную таблицу с дополнительным возможностями для пользователя

## Описание работы
+ Работа выполнена с использованием библиотеки React
+ Верстка выполнена с помощью фреймворкa Bootstrap
+  Реализована сортировка по столбцам: при нажатии на название столбца строки таблицы сортируются по возрастанию, при повторном клике — по убыванию 
+  Реализована клиентская пагинация: данные отображаются постранично, максимум 50 элементов на страницу
+  Реализована пользовательская навигация для перехода по страницам
+  Реализована фильтрация: компонент предоставляет текстовое поле, в которое пользователь может ввести текст и строки таблицы. Перефильтрация осуществляется по нажатию на кнопку "Найти"
+ По клике на строку таблицы значения полей выводятся в дополнительном блоке под таблицей
+  Реализована возможность выбора загрузки данных (32 элемента или 1000 элементов)
